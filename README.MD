# Urban Heat Mitigation using AI/ML

## Bharatiya Antariksh Hackathon 2026

### Problem Statement

Develop a geospatial AI/ML-based system to identify urban heat hotspots, analyze drivers of urban heating, and generate optimized cooling strategies.

### Proposed Solution

Our solution integrates satellite imagery, weather data, and urban infrastructure information to:

- Detect heat hotspots
- Analyze heat drivers
- Predict Land Surface Temperature
- Simulate cooling interventions
- Recommend optimized mitigation strategies

### Technologies

- Python
- GeoPandas
- Rasterio
- Scikit-Learn
- XGBoost
- Streamlit
- Folium

### Expected Outcome

An AI-powered decision support system for urban heat mitigation.
